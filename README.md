Collection_simulator
====================

Visual Collection Simulator written in Java with OpenGL 

**Welcome to the Collection_simulator wiki!**

This is à course project for Computer Graphics in Technical University of Sofia. So far, it has 3 excellent 
marks and lets hope the number will grow. 


The first creator of the project: [Kiril Aleksandrov](https://github.com/kaleksandrov)
